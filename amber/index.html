1. Introduction

分析功能主要包含: Phylogenetic Analysis, Evolutionary Analysis, Gene Set Enrichment Analysis

Phylogenetic Analysis访问「文件/编辑/查看/帮助」等全局功能。
Evolutionary Analysis模块切换（如：项目、数据、分析、报告、设置）。
Gene Set Enrichment Analysis当前模块的主交互区域（表格、图表、表单等）。

2. Phylogenetic Analysis

2.1 Collection

1 创建或打开项目

在菜单栏选择 文件 → 新建项目（或 打开项目）。为项目选择一个空目录，用于保存配置与输出文件。

2 导入数据

进入「数据」模块，点击 导入数据，选择支持的文件格式（如：FASTQ/CSV/JSON 等）。导入后会在数据表中显示摘要信息。

可批量导入；重复导入会自动去重或提示覆盖（具体行为以设置为准）。

2.2 Alignment

1 MAFFT

转到「分析」模块，选择预置模板或自定义参数。重要项示例：

  • 线程数/内存上限（影响性能与稳定性）；
  • 分析模式（如：对齐 / 识别 / 报告）；
  • 结果输出目录（默认为项目目录内的 results/）。
2 MUSCLE

点击 开始。任务面板会显示实时日志、进度条与估计剩余时间。可随时暂停/继续或取消。

长任务建议勾选「完成后通知」，后台运行结束会弹出提醒。
3 Clustal Omega

在「报告」模块查看图表与指标，支持导出为 HTML / PDF / PNG。可点击图例筛选数据系列、在图上框选缩放。

3. Evolutionary Analysis

3.1 Molecular Evolution Analysis

项目信息名称、路径、创建时间、最近修改。
模板管理保存/加载分析模板，便于复用参数。
备份与恢复一键压缩备份项目；崩溃后可自动恢复上次状态。

3.2 Population Genetics Analysis

数据列表展示文件名、大小、导入时间、校验状态。
预览抽样预览前 N 行；可切换原始/解析视图。
校验支持 MD5/SHA256；异常高亮并可重传。

3.3 Community-Ecology Diversity

模式对齐 / 调用 / 汇报（示例，按你的软件实际命名）。
资源线程、内存、临时目录、GPU（如有）。
高级阈值、过滤规则、重试策略、日志级别。

3.4 报告

图表可交互折线/柱状/散点/热力图等。
导出PDF/HTML/PNG，支持水印与模板。
共享生成只读链接（若连接到你的服务端）。

3.5 设置

通用语言、主题、自动更新、匿名使用统计（可关）。
路径默认项目目录、缓存目录、下载目录。
通知桌面通知、声音、完成提醒。

4. Gene Set Enrichment Analysis (GSEA)

以下为默认快捷键，可在「设置 → 快捷键」中自定义：

5. NCBI

  • 检查显卡驱动与系统更新；
  • 尝试在设置中关闭「硬件加速」；
  • 删除缓存目录后重启软件。
  • 确认文件编码与扩展名正确；
  • 查看「数据 → 校验」报错详情；
  • 路径包含中文或空格时，建议移动到英文路径。
  • 在「分析 → 资源」降低线程/内存占用；
  • 确保输出目录在本地 SSD；
  • 查看状态栏错误图标并导出日志给我们。
